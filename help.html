PG MapView Help

This document describes how to use the capabilities of PG-MapView.

PG-MapView is a simple web map client which allows displaying GeoJSON datasets obtained from servers implementing the OGC API for Features.

Add a Layer

  • Click .
  • Enter the following information to specify the layer:
    • URL - the URL of the OAF service
    • Collection - the name of the collection to query. The list of collections provided by the service can be obtained by clicking
  • Optionally, enter layer query parameters:
    • Limit - the maximum umber of features to provide in the response
    • Precision - the number of decimals in GeoJSON ordinates
    • Bbox - the extent of data to query. Click to use the extent of the current map view.
    • Transform - one or more functions to use to transform the response geometry
  • Click to create the new layer, or to abort.

Working with Layers

  • Click the checkbox to turn the layer on or off
  • Click the color button to change the color of the layer
  • Click the layer name to show the Layer Control panel

Layer Control panel

  • X - deletes the layer from the map
  • i - shows the layer info box
  • Z - zooms to the layer extent
  • R - requeries the layer. Shift-click uses the current map extent.
  • S - change the layer settings
